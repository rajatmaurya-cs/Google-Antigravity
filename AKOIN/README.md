## Project Overview

This project is a responsive homepage and brand concept for the **Institute of Digital Risk (IDR)**, an industry-led training and deployment institute focused on digital, cyber, and AI risk.

The objective of the assignment was to design a clean, professional brand identity and develop a responsive single-page website explaining the instituteвЂ™s mission, services, and community.

The website is designed to work across **desktop and mobile devices**, following modern UI principles and accessible design standards.

---

## Features Implemented

### 1. Brand & Logo Design

* Cube-inspired geometric logo representing **structure, resilience, and digital risk frameworks**
* Color palette: **Orange, White, and Black** for a modern technology-focused identity
* Two logo variants:

  * Icon-only version
  * Icon + text: *Institute of Digital Risk*

---

### 2. Responsive Homepage Sections

#### Hero Section

* Mission-focused headline
* Short description explaining IDRвЂ™s purpose
* Call-to-action button

#### About IDR

* Explains IDR as an industry-led training and deployment institute
* Highlights collaboration between **academic research and real-world industry practice**

#### Service Pillars

Three key operational areas:

1. **Academy**
   Training programs and bootcamps for students and professionals.

2. **Innovation & Incubation**
   Development of new digital risk models, governance frameworks, and AI risk research.

3. **Advisory Services**
   Helping organisations implement frameworks such as **NIST, ISO 27001, and NIS2**.

#### Community Section

Focuses on:

* Students
* Early-career professionals
* Practitioners in sectors such as financial services and critical infrastructure.

#### Contact / Call to Action

A simple contact form allowing users to submit:

* Name
* Email
* Message

## Technical Implementation

The website was built using **vanilla web technologies** as required:

* **HTML5** (semantic structure)
* **CSS3** (Flexbox/Grid responsive layout)
* **JavaScript** (basic interactivity)

Key technical features include:

* Semantic HTML structure
* Sticky navigation bar
* Smooth scrolling between sections
* Responsive design for mobile and desktop
* Accessible color contrast
* Button hover states and interactive navigation

No CSS frameworks were used, in accordance with the assignment requirements.
